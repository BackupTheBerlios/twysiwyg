translate

public static String translate(String sText,
                               PrintWriter errout,
                               boolean showWarnings)
Méthode principale de la classe qui permet de lancer la traduction d'un fichier au format html. Le fichier est d'abord parsé par la bibliothèque jtidy pour corriger les erreurs syntaxique puis il est traduit en twiki.
Parameters:
sText - Texte à traduire en syntaxe twiki.
errout - Flux où sera affiché les erreurs de syntaxe html. Si à null, alors par défaut,
showWarnings - Permet de spécifier si on veut que les warnings soient affichés ou pas dans errout. les erreurs seront affichée sur le flux standard d'erreur.
Returns:
Une string représentant le contenu du fichier traduit en twiki.
Throws:
FileNotFoundException - Levé si le chemin du fichier 'filePath' n'est pas valide.
IOException - Levé si un problème survient lors de l'initialisation du parsing.
