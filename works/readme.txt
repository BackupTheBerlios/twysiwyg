There are 4 directories in this "works" module :

- twe :
A WYSIWYG editor for TWiki based-on Java Web Start & SOAP Web Services technologies.
Development stopped becauseof Java 1.4 bugs in Swing HTML API.

- refactoring_tool :
A refactoring tool for TWiki based on Java Web Start & SOAP Web Services technologies.
Not finished.

- soap_service :
SOAP interface between Java Web Start launched tools (WYSIWYG & Refactoring Tool) and TWiki.

- socket_service :
Prototype. New version of this interface based on simple sockets communications.
This interface manage "general" & WYSIWYG services only (refactoring specific services are not implemented here).
